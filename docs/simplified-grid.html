Grid

Grid allow us to arrange our content efficiently using rows and columns which was used to be a headache before grid was introduced. here we have two basic grid components:

1. Two column grid

2. Three column grid

1. Two Column Grid:

            
                <div class="grid-container grid-columns-two">
                    <div class="child1"></div>
                    <div class="child2"></div>
                </div>

2. Three Column Grid:

            
                <div class="grid-container grid-columns-three">
                    <div class="child1"></div>
                    <div class="child2"></div>
                    <div class="child2"></div>
                </div>
