Avatar

The Avatar is used to represent a user, and displays the profile picture. Here, we have 3 avatar components:

1. Basic avatar icon

2. Fallback: In case user pic is not available

3. Avatars with different sizes

1. Basic avatar icon:

            
                <div class="avatar-container">
                    <img class="avatar avatar-one" src="pic source">
                    <img class="avatar avatar-two" src="pic source">
                    <img class="avatar avatar-three" src="pic source">
                    <img class="avatar avatar-four" src="pic source">
                    <img class="avatar avatar-five" src="pic source">
                </div>
